# Prompt Engineering

## 📌 Descripción
Este documento recoge las mejores prácticas para crear prompts efectivos al usar herramientas de IA.

## 🧠 Qué se documentará aquí
- Cómo escribir buenos prompts
- Ejemplos de prompts utilizados
- Errores comunes
- Técnicas para mejorar resultados

## 🎯 Objetivo
Mejorar la calidad de las respuestas generadas por IA.
